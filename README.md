# SK 海力士 ADR 价格速算

一个无需安装第三方依赖的本地网页小程序。启动后会自动获取：

- Nasdaq `SKHY` ADR 美元价格；
- 美国东部时间 20:00–04:00 的 `BOATS:SKHY` 夜盘价格；
- KOSPI `000660` 韩元价格；
- SK 海力士韩股当日外资券商盘中估算流向；
- `USD/KRW`（1 美元兑韩元）汇率。

程序把一股韩股换算成美元，再按 ADR 比例折算为一份 ADR 的理论美元价格，并显示美股 ADR 的溢价或折价。

页面还包含：

- 最近约 20 个已确认交易日的外资净买卖柱状图和每日明细；
- SKHY 上市以来可对齐交易日的 ADR 历史溢价率图；
- 历史溢价率会随页面中的 ADR 比例输入即时重算；
- 最近约 20 个发布日的韩国全市场信用融资余额和投资者预托金表。

## 一键运行

macOS：双击 `启动SK海力士速算.command`。

也可以在终端运行：

```bash
./run.sh
```

浏览器会自动打开 `http://127.0.0.1:8787`。关闭启动脚本的终端窗口，或按 `Control+C`，即可停止程序。

只要求电脑已安装 Python 3；macOS 通常可直接使用。程序仅监听本机地址，行情和输入不会上传到本程序之外。

## 计算方式

```text
一股韩股美元价 = 韩股价格（KRW）÷ USD/KRW
一份 ADR 理论价 = 一股韩股美元价 ÷ 每股对应 ADR 份数
溢价率 =（ADR 市价 ÷ ADR 理论价 − 1）× 100%
```

官方发行资料显示每份 ADR 代表 `0.1` 股韩股，即默认 `1 股韩股 = 10 份 ADR`。页面允许修改这个比例，因此也可以输入 `11` 做自定义假设速算。

## 数据源与限制

- ADR 常规盘：优先 Nasdaq；失败时尝试 Yahoo Finance。
- ADR 夜盘：TradingView / Blue Ocean ATS（BOATS）实时成交价。
- 韩股和汇率：优先 Naver Finance；失败时尝试 Yahoo Finance。
- 韩股外资流向：Naver Finance“投资者买卖趋势”中的当日前五家外资券商席位估算。
- 韩国市场资金：Naver Finance“证券市场资金趋势”；口径对应韩国金融投资协会的投资者预托金和信用融资余额，单位为亿韩元。
- 夜盘开放时，溢价率自动使用 BOATS 夜盘价；夜盘关闭或取数失败时自动回退常规盘/最新可得价。
- 外资流向约延迟 20 分钟，会显示估算买入、估算卖出和净流向；它不是 KRX 收盘后的最终外资净买卖数据。
- 免费行情可能存在延迟；韩美市场的开闭市时间也不同。
- BOATS 是独立 ATS 夜盘市场，流动性通常较低，报价和价差可能与 Nasdaq 常规盘明显不同。
- 计算未计入 ADR 托管费、税费、汇兑点差及跨境转换限制，不构成投资建议。

历史溢价率按同一个日历日的 Nasdaq、韩股和 USD/KRW 收盘值计算。由于韩国市场先收盘、美股后收盘，该序列适合观察相对定价变化，不代表三个价格可以同时成交。SKHY 刚上市时历史点数会比较少，之后会随交易日自动增加。

融资余额和投资者预托金是韩国全市场统计，不是 SK 海力士单一股票的融资数据；发布日期通常落后于当前交易日。

## 部署到 GitHub Pages

仓库包含 `.github/workflows/deploy-pages.yml`。它会在推送到 `main`、手动触发以及每 5 分钟定时触发时：

1. 联网生成最新 `snapshot.json` 和 `history.json`；
2. 把纯静态网页打包；
3. 部署到 GitHub Pages。

首次使用时，在 GitHub 仓库打开 `Settings → Pages`，把 `Source` 设为 `GitHub Actions`。GitHub Free 用户需要使用公开仓库；成功后页面地址通常是：

```text
https://<你的用户名>.github.io/<仓库名>/
```

GitHub Actions 的定时任务最快为 5 分钟一次，并可能排队延迟；外资盘中估算本身约延迟 20 分钟。因此公开站点是“定时更新”，不是逐笔行情服务器。

公开页面上的“读取已发布数据”只重新读取 Pages 当前版本，不会直接访问行情源。仓库管理员需要立即更新时，可点击“管理员立即更新”，在打开的 GitHub Actions 页面选择 `Run workflow`；原页面会轮询新版本，并在部署完成后自动显示新的发布时间。GitHub 要求操作者已登录且拥有该仓库的写权限。

本地可预先验证 Pages 构建：

```bash
python3 build_pages.py --output dist
python3 -m http.server 8000 --directory dist
```
